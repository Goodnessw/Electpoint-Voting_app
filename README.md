# Electpoint - Voting App

## Project Overview

The **Voting App** is a modern, web-based application that allows users to create polls, vote on options, and view real-time results. This project is built with **React**, **TypeScript**, and **Tailwind CSS**, making it fast, responsive, and easy to maintain.  

It is designed for scalability and can be deployed on any static hosting service or server that supports Node.js.

---

## Live Demo

You can access the live version of the application here:  
**URL**: `https://electpoint.vercel.app/admin`  

---

## Features

- **Create elections:** Users can create elections.  
- **Vote:** Users can cast votes and the app prevents multiple voting from the same user/session.  
- **Real-time Results:** results update dynamically as votes are cast.  
- **Responsive Design:** Works on desktop, tablet, and mobile devices.  
- **User-friendly Interface:** Built with modern UI components and Tailwind CSS for a clean look.  

Optional Features You Can Add:

- Authentication for user accounts
- Admin panel for managing Votes
- Analytics for voting trends



---

## Technologies Used

- **React** – Frontend library for building UI components.  
- **TypeScript** – Ensures type safety and maintainable code.  
- **Vite** – Fast development server and build tool.  
- **Tailwind CSS** – Utility-first CSS framework for responsive styling.  
- **shadcn-ui** *(optional)* – Modern React UI component library.

## Installation & Setup

Follow these steps to run the project locally:

```bash
# 1. Clone the repository
git clone <https://github.com/Goodnessw/Electpoint-Voting_app.git>

# 2. Navigate into the project directory
cd <voting-app>

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open your browser and go to the port shown in your terminal. The app will auto-reload when you make changes.

---

## Building for Production

To build the app for deployment:

```bash
npm run build
```

This will generate a **`dist/`** folder containing optimized files ready for hosting.

---

## Deployment

You can deploy this app to any static hosting service:

- **GitHub Pages**  
- **Netlify**  
- **Vercel**  
- **Firebase Hosting**  

Simply upload the contents of the `dist/` folder to your chosen platform and configure your domain if needed.
